# ⚡ Hyke - Multi-Agent AI SaaS

### 🚀 A Próxima Geração de Gestão de Agentes de IA

**Hyke** é uma infraestrutura completa para empresas que buscam escalar seu atendimento e operações com Inteligência Artificial. 

- **Frontend**: Dashboard administrativo premium em Next.js 16.
- **Backend/ORM**: PostgreSQL + Prisma.
- **Automatização**: Orquestração via n8n.
- **Integração**: Conectividade nativa com Evolution API (WhatsApp).

#### 🌟 Destaques Técnicos
- **Multi-tenant Core**: Arquitetura pronta para múltiplos clientes isolados.
- **Dynamic Prompts**: Gestão individual de contexto e tom de voz por agente.
- **Subscription Ready**: Suporte a Stripe e Mercado Pago integrado.
